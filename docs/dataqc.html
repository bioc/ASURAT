Introduction to ASURAT (version 0.0.0.9000)

Chapter 3 Data quality control (QC)

In this section, we remove the genes and cells with low quality, by processing the following three steps: (1) quick gene filtering based on expression profile across cells, (2) cell filtering based on QC metrics, and (3) second gene filtering based on mean expression levels across cells.

The following function do_quickQC() removes the genes for which the number of non-zero expressing cells is less than min_nsamples. In addition, this function counts the number of reads, the number of non-zero expressing genes, and the percent of reads that map to the mitochondrial genes, e.g., MT-Co1, mt-Co1, etc.

day1_norm <- do_quickQC(obj = day1_norm, min_nsamples = 10,
                        mitochondria_symbol = "^MT-")
day7_hypo <- do_quickQC(obj = day7_hypo, min_nsamples = 10,
                        mitochondria_symbol = "^MT-")
sc68_vehi <- do_quickQC(obj = sc68_vehi, min_nsamples = 10,
                        mitochondria_symbol = "^MT-")
sc68_cisp <- do_quickQC(obj = sc68_cisp, min_nsamples = 10,
                        mitochondria_symbol = "^MT-")
pbmc_4000 <- do_quickQC(obj = pbmc_4000, min_nsamples = 10,
                        mitochondria_symbol = "^MT-")
pbmc_6000 <- do_quickQC(obj = pbmc_6000, min_nsamples = 10,
                        mitochondria_symbol = "^MT-")

The results are stored in obj[["sample"]][["nReads"]], obj[["sample"]][["nGenes"]], obj[["sample"]][["percent_MT"]], and obj[["data"]][["raw"]].

Users can always check the parameter values in obj[["history"]]. For example, obj[["history"]][["do_quickQC"]][["min_nsamples"]].

The following function plot_metaData() outputs a scatter plot between the input meta data for each cell The arguments are df (two-dimensional data frame), title (title of the plot), xlabel (x-axis label), and ylabel (y-axis label).

my_func <- function(obj, metaname1, metaname2){
  df <- data.frame(x = obj[["sample"]][[metaname1]],
                   y = obj[["sample"]][[metaname2]])
  p <- plot_metaData(df = df,
                    title = obj[["history"]][["make_asurat_obj"]][["obj_name"]],
                    title_size = 16, xlabel = metaname1, ylabel = metaname2)
  return(p)
}

The following function plot_metaData_wAnnotation() outputs a scatter plot for the meta data input, which is ranked according to their values. The arguments are vec (one-dimensional vector of single-cell QC metrics, e.g., one of obj[["sample"]][["nReads"]], obj[["sample"]][["nGenes"]]), target_xRange (x-axis interval in which one would like to find an inflection point), n (number of dividing points for target_xRange), title, xlabel, and ylabel. Note that the inflection points are used for the cell filtering below. Note that the calculated inflection points are approximated to the nearest integers.

my_func <- function(obj, metaname, target_xRange){
  vec <- obj[["sample"]][[metaname]]
  p <- plot_metaData_wAnnotation(
    vec = obj[["sample"]][[metaname]], target_xRange = target_xRange,
    n = round(0.1 * (target_xRange[2] - target_xRange[1])),
    title = obj[["history"]][["make_asurat_obj"]][["obj_name"]],
    title_size = 16, xlabel = "Ranked samples", ylabel = metaname
  )
  return(p)
}
# ----------------------------------------
# day1_norm
# ----------------------------------------
target_xRange <- c(100, 10000)

The following function plot_meanReads_wAnnotation() outputs a scatter plot for mean of number of reads for each gene, which is ranked according to their values. The arguments are obj, threshold (set by trial and error), digits (number of digits after the decimal point for the annotated value), title, xlabel, and ylabel.

my_func <- function(obj, threshold){
  p <- plot_meanReads_wAnnotation(
    obj = obj, threshold = threshold, digits = 2,
    title = obj[["history"]][["make_asurat_obj"]][["obj_name"]],
    title_size = 16, xlabel = "Ranked genes", ylabel = "Mean of raw read counts"
  )
  return(p)
}
# ----------------------------------------
# day1_norm

p <- my_func(obj = day1_norm, threshold = 0.2)
filename <- "figures/figure_01_0030.png"
