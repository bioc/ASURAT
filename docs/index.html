Introduction to ASURAT (version 0.0.0.9001)

Keita Iida

2021-09-09

Chapter 1 Overview

ASURAT is an analysis pipeline, which simultaneously performs unsupervised clustering and biological interpretation in terms of cell type, disease, biological process, and signaling pathway activity. Inputting a single-cell RNA-seq (scRNA-seq) data and knowledge-based databases, ASURAT transforms the read count table into an original multivariate table, termed a sign-by-sample matrix (SSM). By analyzing SSMs, users can cluster cells to aid their interpretation.
