2021-07-01

Chapter 1 Overview

ASURAT is a single-cell RNA sequencing (scRNA-seq) data analysis pipeline, developed for simultaneously clustering single-cell transcriptomes and biological interpretation. Inputting an scRNA-seq data and knowledge-based databases (DBs), ASURAT creates lists of biological terms with respect to cell type and biological functions. Then, ASURAT creates a functional spectrum matrix. By analyzing this matrix, users can cluster cells to aid their interpretation.
